# medical chatbot

# Complete medical chatbot with LLM, Langchain, Pinecone, Flask and AWS

# Steps to Run this Project

## Steps:


### Step 01: clone the Repository

Clone the Repository
'''bash
git clone https://github.com/ASHEESHPATEL1996/medical_chatbot.git
'''

### Step 02: Create the virtual Environment and activate it

Create the virtual Environments
'''bash
python -m venv .venv
or
conda create -n .venv python=3.13.5 -y
'''

activate the environment
'''bash
.venv\Scripts\activate
or 
conda activate .venv
'''

### Step 03: install the Requirements.txt
'''bash
pip install -r requirements.txt
'''
